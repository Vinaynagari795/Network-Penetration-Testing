# Network Penetration Testing

A collection of practical network penetration testing labs focused on service enumeration, remote access assessment, authentication testing, and security analysis performed in a controlled lab environment.

This repository documents hands-on exercises used to identify, enumerate, and assess common network services and protocols frequently encountered during penetration testing engagements.

---

## Repository Objectives

- Develop practical network penetration testing skills.
- Understand service enumeration methodologies.
- Assess network-exposed services.
- Identify authentication weaknesses.
- Document findings and remediation strategies.
- Build a professional cybersecurity portfolio.

---

## Projects

| Project | Description |
|----------|-------------|
| FTP Enumeration & Security Assessment | Enumeration and security assessment of FTP services. |
| SMB Enumeration & Security Assessment | SMB share discovery, enumeration, and security analysis. |
| SNMP Enumeration & Security Assessment | Information gathering through SNMP enumeration. |
| NFS Enumeration & Security Assessment | NFS share discovery and access assessment. |
| MSSQL Enumeration & Security Assessment | MSSQL service enumeration and authentication testing. |
| WinRM Enumeration & Security Assessment | Windows Remote Management enumeration and access testing. |
| RDP Enumeration & Security Assessment | Remote Desktop Protocol enumeration and security assessment. |
